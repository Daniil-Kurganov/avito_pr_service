# Тестовое задание (автор Курганов Даниил Андреевич)
## Вопросы и самостоятельно приянтые решения
### Запуск
В условиях тестового задания не обнаружил требование к запуску, поэтому, принял решение, что сервис будет доступен по сокету "127.0.0.1:8080".
### Логгирование
Логирование инициализируется на уровне "Info" и выводится в консоль.
### Ответы сервера, не указанные в API
При возникновении ошибки обработки HTTP-запроса, к примеру возникновение ошибки в процессе валидации значений запроса, сервер ответит кодом ошибки, а информация об ошибке будет выведена в логах на уровне "Error".
