## Вопросы и самостоятельно приянтые решения
### Логгирование
Логирование инициализируется на уровне "Info" и выводится в консоль.
### API и валидация запросов
При реализации, опираюсь на описанное API и что такие ситуации как запрос на добавление команды с пустым составом не будет поставляться на вход. Иначе, сервер будет отправлять HTTP-ответ, содержащий только код.
